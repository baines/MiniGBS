# MiniGBS

MiniGBS is a small .gbs file (gameboy music) player with an ncurses UI.
It's not cycle-accurate, but still sounds good for most of the files I've tested.

## Hotkeys
	n/→     Next song
	p/←     Prev song
	r/↑     Restart song
	space/↓ Pause/Resume
	-/=     Volume down/up
	1/2/3/4 Toggle channel
	esc/q   Quit
